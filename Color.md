# Haven Estate Color Palette

This document serves as a reference for the official color palette used in the Haven Estate application. Use these colors consistently throughout the project for a cohesive visual experience.

## Color Scheme

### Primary Color
- **Hex**: #235C47 (Rich forest green)
- **Usage**: 
  - Primary buttons and CTAs
  - Main brand elements
  - Headers and important text
  - Accent elements
  - Hover states for secondary elements

### Secondary Color
- **Hex**: #F9F7F6 (Soft neutral)
- **Usage**: 
  - Backgrounds for sections
  - Card backgrounds
  - Hover states for primary elements
  - Borders and dividers
  - Subtle backgrounds

### Third Color
- **Hex**: #FFFFFF (Clean white)
- **Usage**: 
  - Main background color
  - Text on dark backgrounds
  - Card backgrounds when needed for contrast
  - Background for images

## Usage Guidelines

### Background Colors
- Use **#FFFFFF** as the main background for pages
- Use **#F9F7F6** for section backgrounds to create visual separation
- Use **#235C47** for accent sections or when high contrast is needed

### Text Colors
- Use **#235C47** for headings and primary text
- Use appropriate opacity or contrast for text on colored backgrounds
- Use **#FFFFFF** for text on dark backgrounds

### Image Backgrounds
- Use **#F9F7F6** as a background for images when needed for consistency
- Use **#FFFFFF** as a fallback for image containers
- Ensure images have proper contrast with surrounding elements

### Interactive Elements
- Primary buttons: **#235C47** with **#FFFFFF** text
- Secondary buttons: **#F9F7F6** with **#235C47** text
- Hover states: Darken colors by 10-15% for visual feedback

## Accessibility
All color combinations in this palette have been selected to meet WCAG 2.1 AA contrast standards. When using these colors, ensure sufficient contrast between text and background elements for readability.
